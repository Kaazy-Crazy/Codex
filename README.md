# マルシェ来場者チェックイン

Express とシングルページ UI で構成された簡易的な来場者チェックインアプリです。サーバーはメモリ上で来場者情報を管理します。

## セットアップと起動
1. 依存パッケージをインストールします。
   ```bash
   npm install
   ```
2. サーバーを起動します。
   ```bash
   npm start
   ```
3. ブラウザで `http://localhost:3000` を開きます。

## 使い方
- 受付フォームに氏名・連絡先・区分・同行者人数・メモを入力し、「リストに追加」を押すと新規来場者が登録されます。
- リスト右上の検索ボックスで氏名やメールアドレスを検索できます。
- ステータス切替ボタンで「すべて / 未入場 / チェックイン済み」のフィルタリングが行えます。
- 各カードの「チェックイン」ボタンで入退場をトグルできます。削除ボタンでリストから除外できます。
- 画面上部の統計カードに合計数・チェックイン数・未入場数がリアルタイムに反映されます。

## 注意事項
- データはサーバーのメモリ上に保持されるため、再起動するとリセットされます。
- バックエンド API は `server.js` に実装されており、必要に応じて永続化や認証などを追加してください。
